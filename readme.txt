Description:
The application is an online service that allows users to extract textual content from a specified URL, automatically process it to identify key topics, and then generate texts conforming to various formats and guidelines, including SEO, blogging, journalistic headlines, school/university, using pre-formatted prompts.

Main Features:
Automatic extraction of textual content from a URL.
Identification and generation of key topics based on the extracted content.
Generation of texts conforming to different formats and guidelines.
Intuitive and user-friendly interface.

Technologies Used Currently:
HTML
JavaScript
Node.js

How to Use:
Open the application in the browser.
Enter the URL of the website from which you want to extract textual content.
Click on a button to start the processing.
Select the most relevant content and the desired format for the text.
View the generated text with real-time editing capabilities.
Print and download function in PDF/Word format.
Creation of a personal archive with searches performed and texts generated.

Examples of Use:
SEO Scenario: Generate search engine optimized texts using relevant keywords.
Blog Scenario: Create original blog articles based on specific topics extracted from other websites' content.
Journalistic Headline Scenario: Produce news articles, including technical ones, based on relevant information from websites.
School/University Scenario: Create summaries or presentations on study topics using information from online sources.

If you want to run the application locally:
Make sure you have Node.js installed on your system.
Clone this repository.
Navigate to the cloned repository directory via the terminal.
Run the npm install command to install dependencies.
Start the application using the npm start command.
Open the browser and visit http://localhost:3000 to access the application.
